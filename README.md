# BirHaber
Bu proje, Mobil Uygulama Geliştirme dersi kapsamında geliştirdiğim ilk mobil uygulamadır. Uygulama, Firebase ve Kotlin kullanılarak tasarlanmıştır.
Uygulama, haber eklenmesi, düzenlenmesi, görüntülenmesi , favorilere eklenmesi ve silinmesi gibi temel işlevlerin yanı sıra, kullanıcı yönetimi özelliklerine de sahiptir. Kullanıcılar üzerinde aktif/pasif durumu değiştirme, kayıt olma, giriş yapma ve şifre sıfırlama gibi işlemler gerçekleştirilebilmektedir.
